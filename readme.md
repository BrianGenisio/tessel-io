# Tessel-IO

[![Build Status](https://travis-ci.org/rwaldron/tessel-io.png?branch=master)](https://travis-ci.org/rwaldron/tessel-io)

## Tessel-IO is compatible with Tessel 2

Tessel-IO is an IO Plugin that enables writing Johnny-Five programs the run on a Tessel 2. This project was built at [Bocoup](http://bocoup.com)
